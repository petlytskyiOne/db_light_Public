Command

Data selection:

Method Code Description
"use" use my_data create a database
createCollection db.createCollection("mycollection") collections are created automatically when you insert the first document
.find db.my_data.find() prints all data from the database "data"
.find db.my_data.find({ege: 22, email:test@test.com}, {_id: 0}) data selection by two indicators
.find "$or" db.my_data.find({$or: [{ege: 21}, {email: "test@test.com"}]}, {_id: 0}) selection of data from two "or"
.find .sort" db.my_data.find({}, {_id:0}).sort({age:1, email:-1}) sort function by "age" field in ascending order 1,descending order -1
.find .limit db.my_data.find({}, {_id: 0}).limit(2) display only the first two records and do not show the id
.find "$or", "$lt" db.my_data.find({$or: [{ege: {$lt: 38}, {email: "test@test.com"}]}, {_id: 0}).sort({age: 1}) $lt: less than number
$gt: more than number
$gte: greater than or equal to this number
$eq: is exactly this number
$ne: is not equal to this number
.find "$in" db.my_data.find({name: {$in: ["jack", "John", "Bob"]}}, {_id: 0}) $in: selection of objects that match these characteristics
$nin: selection of objects that do not match these characteristics

Execution Command Description
Documents can be inserted, updated, and deleted from collections. ddb.mycollection.insert({ name: "John", age: 30, city: "New York" })
Adding a data structure via the terminal db.users.insertOne({"name":John,"email":"petlytskyi@gmail.com","age":23})
Update information: db.mycollection.update({ name: "John" }, { $set: { age: 31 } })
Deleting information: db.mycollection.remove({ name: "John" })
Sorting and limiting: db.mycollection.find().sort({ name: 1 }).limit(5)
Indexation: db.mycollection.createIndex({ name: 1 })
Aggregation: db.mycollection.aggregate([
{ $group: { _id: "$city", total: { $sum: 1 } } }
])
